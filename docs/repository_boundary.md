# Repository Boundary

Use this repository for:
- feature extraction and ML pipeline code,
- analysis configurations,
- generated analysis outputs (locally, not versioned by default),
- manuscript-oriented analysis documentation.

Do not use this repository for:
- final BIDS data publication payloads,
- data descriptor release assets intended for the journal submission repository.

Recommended split:
- Data-descriptor repository: BIDS dataset, metadata, data-centric documentation.
- Analysis repository (this one): reproducible code and model results.
